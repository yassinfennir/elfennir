**The Prompt:**

### 2.1 60-Second TikTok Script Engine
**Category:** Video Scripts
**The Prompt:**
> Write a 60-second TikTok script about [TOPIC] for a [NICHE] creator. Structure: Hook (0-3s) that stops the scroll, Setup (3-15s) that builds tension or context, Value bomb (15-45s) with 3 concrete points, CTA (45-60s) that drives [GOAL: follows/saves/comments/shares]. Include: on-screen text suggestions for each section, background music mood, and B-roll ideas. Writing style: conversational, not scripted-sounding.

**Expected Output:** A timestamped 60-second script with production notes.
**Pro Tip:** Read it out loud. If it sounds like a script, rewrite it until it sounds like you're talking to a friend.

### 2.3 Talking Head Script
**Category:** Video Scripts
**The Prompt:**
> Write a 90-second talking head script about [TOPIC]. I am a [NICHE] creator. The script must: feel like natural speech (contractions, pauses, personality), include 2-3 hand gesture moments, have energy shifts (loud → quiet → impactful), include a personal anecdote or example, and end with a clear CTA. Format: each line as it should be spoken, with [PAUSE], [EMPHASIS], and [GESTURE] cues.

**Expected Output:** A speech-ready script with delivery cues.
**Pro Tip:** The best talking head videos feel unrehearsed. Practice 3 times, then film on the 4th — not the 20th.

## Content Creator Cheat Sheet

### Optimal Posting Times (General — test for YOUR audience):
- TikTok: 7-9 AM, 12-1 PM, 7-9 PM (local time)
- Instagram: 11 AM-1 PM, 7-9 PM (local time)
- YouTube: Thursday-Saturday, 2-4 PM (local time)
- Twitter/X: 8-10 AM, 12-1 PM (local time)

### Ideal Content Lengths:
- TikTok: 15-60 seconds (sweet spot: 30-45 sec)
- Instagram Reels: 15-30 seconds
- YouTube Shorts: 30-58 seconds
- YouTube long-form: 8-15 minutes
- Instagram carousel: 7-10 slides
- Twitter thread: 7-12 tweets

### Engagement Formulas:
- Engagement Rate = (Likes + Comments + Shares + Saves) / Followers x 100
- Good: 3-6% | Great: 6-10% | Viral: 10%+
- Reach Rate = Reach / Followers x 100

### The 80/20 Rule for Creators:
- 80% value content (educate, entertain, inspire)
- 20% promotional content (sell, CTA, pitch)
